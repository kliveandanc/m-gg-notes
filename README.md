# m-gg-notes

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面内容不针对特定域名或网站，主要用于记录、展示或测试各类前端技术、交互效果与静态内容。

仓库以纯静态文件形式组织，所有页面均可直接在浏览器中打开查看，无需后端服务支持。

## 目录说明

```
m-gg-notes/
├── pages/        # 存放所有独立 HTML 页面
├── assets/       # 公共资源文件（图片、样式、脚本等）
└── README.md     # 本文件
```

- `pages/` 目录下每个 HTML 文件均为独立页面，命名规则为 `page-xxx.html`，便于区分和引用。
- `assets/` 目录用于存放多个页面可能共享的静态资源，如 CSS、JavaScript 或图片文件。

## 页面归档说明

- 每个 HTML 页面均为自包含的独立文档，可单独打开使用。
- 页面内容涵盖但不限于：信息展示、交互原型、功能测试、样式演示等。
- 所有页面均不依赖外部网络资源，除非在页面内显式说明。
- 页面版本通过文件名或页面内的注释进行标识，便于追溯和更新。

## 维护说明

- 如需新增页面，请在 `pages/` 目录下创建新的 HTML 文件，并遵循命名规范。
- 若页面使用共享资源，请将资源文件放置于 `assets/` 对应子目录中。
- 本仓库不提供自动化构建或部署流程，仅作为静态文件存档使用。
- 欢迎提交 Issue 或 Pull Request 来报告问题或建议改进。
